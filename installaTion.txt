Docker Desktop
>Check docker install : cli -> "docker"

Build docker image :docker build -t flask-docker-demo .
Run the container :docker run -p 5000:5000 flask-docker-demo 
Tag your image: docker tag flask-docker-demo asang/flask-docker-demo:latest1
Assure that you are logged in :docker login
Push image to dockerhub :docker push asang/flask-docker-demo:latest1
Pull image from dockerhub: docker pull asang/flask-docker-demo:latest1
Run the Pulled image: docker run -p 5000:5000 asang/flask-docker-demo:latest1
