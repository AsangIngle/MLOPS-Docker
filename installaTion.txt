Docker Desktop
>Check docker install : cli -> "docker"
>Pull the hello-world image and run it 

Build docker image :docker build -t flask-docker-demo .
Run the container :docker run -p 5000:5000 flask-docker-demo 
Tag your image: docker tag flask-docker-demo asang55/flask-docker-demo:latest
Assure that you are logged in :docker login
Push image to dockerhub :docker push asang55/flask-docker-demo:latest
Pull image from dockerhub: docker pull asang55/flask-docker-demo:latest
Run the Pulled image: docker run -p 5000:5000 asang55/flask-docker-demo:latest
